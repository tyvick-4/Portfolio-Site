## THE PROBLEM
Discord has 656M registered users but only 259M MAU and 31.5M DAU (12% DAU/MAU — below a16z's 25% "OK" threshold). ~400M registered users are dormant. 99.9% of Discord's content is blocked from search engines (robots.txt: `Disallow: /channels/`). Only ~32K of 28.4M servers are publicly discoverable. Discord has 0% AI citation share. Meanwhile, Reddit rode open content to #3 US website, 68% search-sourced traffic, #1 AI citation ranking, $1.3B revenue, and $203M+ in AI licensing deals.

## THE OPPORTUNITY
**Selective openness**: Make Forum Channels (already structured like forums) indexable with admin opt-in. This unlocks millions of pages of high-quality community content for search engines and AI models — without exposing private real-time chat. Conservative estimate: 3M reactivated users/quarter from search-driven re-entry alone.

## WHY USERS LEAVE (5 personas)
1. **Gamers (74% of servers)**: Activity tied to game lifecycle. Go dormant between releases. Return for new games.
2. **Server Admins**: Moderator burnout. Discord's own docs describe it. Need growth tools/metrics as motivation.
3. **Non-Gamers (growing to 55%+)**: "Wall of Text Fatigue" → Mute → Forget → Churn. Discovery Deficit: "Discord feels like a dark room."
4. **Privacy-Conscious**: Age verification backlash (Feb 2026), AI feature resistance. Discord expects user number hit.
5. **International (34% APAC)**: Language/timezone barriers. Reddit's intl logged-out growth: +54% YoY.

**Cross-cutting**: Discord is a "knowledge black hole" — valuable answers lost in ephemeral chat. No passive re-discovery path exists.

## THE THESIS: SEARCH = AMBIENT REACTIVATION
Push reactivation (email/notifications) fails because users already muted everything. Pull reactivation (search) works because dormant users naturally encounter their community's content while searching for something else. Reddit proved this: 55.4% of daily traffic is logged-out users arriving via search. 88% of Discord users say they'd return — they just need a trigger. Search provides that trigger without spam.

## KEY METRICS TO OWN
- **Acquisition**: Search impressions, CTR, new registrations from search, server join rate from organic
- **Reactivation**: Resurrection rate (target: 15%/quarter), search-driven resurrection rate, D7/D30 retention of reactivated users
- **Engagement**: DAU/MAU improvement from 12% baseline, session depth of search-referred users
- **GEO**: AI citation frequency, gaming AI answer share, branded search volume
